# CFA ATM terminal app

<h1>Project Name:</h1>

ATM Terminal App

<h2>Description:</h2>

Create an ATM app which does the following:
  - multiple methods
  - utilizing a loop
  - getting user input at multiple stages of the app
  - ability to set an initial balance
  - ability to withdraw
  - ability to deposit
  - ability to exit the ATM if nothing more needs to be done
